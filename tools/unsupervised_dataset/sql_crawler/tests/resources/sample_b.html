This is another sample website with HTML queries.

There will be queries within the HTML, like this.

SELECT a.col, b.col FROM table_a AS a, table_b AS b WHERE a.col = b.col;

SELECT a.col, b.col FROM table_a AS a, table_b AS b;

There will be other queries as well with different keywords.

CREATE TABLE Sample (
  ColumnA int,
  ColumnB varchar(255)
);

Hyperlinks like this should work.

Even if there are SELECT or other keywords in the middle of text, hopefully it should not think these are queries.

SELECT a.col, b.col FROM table_a AS a, table_b AS b WHERE a.col = b.col;

If there are multiple queries within the same tag, it should not be a problem.

ALTER TABLE Sample ADD ColumnC varchar(255);
SELECT a.col, b.col FROM a, b WHERE a.col = 5;

If the query is after a normal sentence, it should still work. SELECT c.col FROM table_c as c;

SELECT statements look like this: SELECT d.col from table_d as d;
